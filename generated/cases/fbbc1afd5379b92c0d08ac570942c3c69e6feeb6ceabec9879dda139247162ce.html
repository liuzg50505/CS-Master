RELATIONAL MODELING REVISIT
Challenge: develop a theory and list guidelines to assess a relational model in
terms of (performance metrics):
пӮ— goodness, i.e., whether the attributes (e.g., PK, FK) form a good relation;
пӮ— quantify the degree of goodness;
пӮ— what pitfalls exist;
пӮ— efficiency in insert/delete/update operations;
пӮ— convey as much information as possible minimizing redundancy (minimize

GUIDELINES FOR A GOOD DESIGN
Guideline 1: The attributes of a relation should make sense
пӮў Attributes of different entities, e.g., students, employees, courses,
should not be in the same relation.
пӮ— Objective: minimize the similarity between relations!
пӮў Any relationship between relations should be represented only
through Foreign Keys & Primary Keys
пӮ— e.g., an employee works on a specific project for 2 hours:
пӮ— WORKS_ON(SSN (FK), PNO(FK), Hours).

THEORY OF FUNCTIONAL DEPENDENCY
Functional Dependency (FD) is a formal metric of the degree of
goodness of relational schema:
пғҳ FD is a constraint derived from the relationship between attributes
вҖңGiven a relation, an attribute X functionally determines an attribute
Y, if a value of X determines a unique value for Y.вҖқ, Codd, 1970
In other words: give me a value of X and IвҖҷll tell you which is the value
of Y in a specific tuple! (X determines Y)
14
THEORY OF FUNCTIONAL DEPENDENCY
пӮў FD: X вҶ’ Y (X uniquely determines Y) holds if whenever two tuples
have the same value for X, they must have the same value for Y
Definition: for any two tuples t1 and t2:
If t1[X] = t2[X] then  t1[Y] = t2[Y]
X вҶ’ Y in R specifies a constraint on all instances, i.e., principle.

FUNCTIONAL DEPENDENCY PRINCIPLES
Lemma 1: If K is a Candidate Key, then K functionally determines
all attributes in relation R, i.e.,
FD: K вҶ’ {R}
Lemma 2: William W. ArmstrongвҖҷs inference rules for FDs (1974):
пӮ— (Reflexive) If Y вҠҶ X then X вҶ’ Y
пӮў X = {SSN, Ename}; X пғ  {SSN}, X пғ  {Ename}
пӮ— (Augmentation) If X вҶ’ Y then X U {Z} вҶ’ Y U {Z}
пӮ— (Transitive) If X вҶ’ Y and Y вҶ’ Z then X вҶ’ Z
