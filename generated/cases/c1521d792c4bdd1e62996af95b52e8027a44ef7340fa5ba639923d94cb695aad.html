What is pandas?
▪ One of the most popular library that data scientists use.
▪ Created by Wes McKinney in 2008, now maintained by Jeff Reback and
many others.
– Author of the textbook: Python for Data Analysis
▪ Panda provides fast, ﬂexible, and expressive data structures: Series and
dataFrames,  designed to make working with “relational” or “labeled” data
both easy and intuitive.
▪ It aims to be the fundamental high -level building block for doing practical,
real world data analysis in Python.
▪ Pandas builds on top of Numpy to ease managing heterogeneous data
sets.
